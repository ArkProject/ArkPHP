ArkPHP
=======
ArkPHP 是一个免费开源的，快速、简单的面向对象的轻量级PHP开发框架。
当前版本 v1.0.1 build 0

##支持
  PHP v5.3以上（开发版本，未测试其他版本）

##源码架层次结构

    src
        framework
            ark.php
            functions.php
            core
                application.class.php
                controller.class.php
                module.class.php
                module
        content
            css
            scripts
            images
        plugs
        app_data
            config.php
        locales
            en-US.php
            zh-CN.php
        modules
            default
                app.php
                controllers
                models
                views
        index.php
    doc
        readme.txt
    build
        arkphp-{version}-{encoding}.rar //版本规则
